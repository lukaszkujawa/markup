# Documentation Files

This folder contains markdown documentation that is imported into Markup on first run.

## Files

- **Welcome.md** - Introduction and overview of Markup
- **D2Lang.md** - Quick start guide for D2 diagram language

## Build Process

These files need to be copied to `public/docs/` for the app to load them. Run:

```bash
cp docs/*.md public/docs/
```

This is required before building or running the app for the first time.
