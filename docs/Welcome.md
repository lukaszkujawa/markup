# Welcome to Markup

**Markup** is a clean, modern markdown editor built for focused writing and visual documentation.

## What is Markup?

Markup combines the simplicity of markdown with powerful real-time preview and diagram capabilities. Write in plain text, see beautiful results instantly.

## Core Capabilities

- **Real-time Preview** - See your markdown rendered as you type
- **Syntax Highlighting** - Code blocks with JavaScript, TypeScript, and more
- **D2 Diagrams** - Create beautiful diagrams from simple text syntax
- **Organized Notes** - Manage notes with folders and drag-and-drop
- **Formatting Toolbar** - Quick access to bold, italic, headings, links, and tables
- **Auto-save** - Your work is automatically saved locally
- **Flexible Layout** - Toggle editor, preview, or file navigation panels

## Quick Start

1. **Create a Note** - Click the `+` button in the top navigation
2. **Write Markdown** - Use the formatting toolbar or type markdown syntax
3. **Preview Live** - See your formatted content update in real-time
4. **Organize** - Create folders and drag notes to organize your work

Start writing and let Markup handle the rest!
